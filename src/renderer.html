🔑 LLM API 발급 방법
💡 최소 1개는 필수입니다. 여러 개 등록하면 장애 시 자동 폴백돼요.
추천: OpenAI GPT-4o-mini (품질·속도 균형)
⭐ OpenAI · GPT-4o-mini (추천 · 유료, 가장 저렴)
1. platform.openai.com/api-keys 접속 → 로그인
2. [+ Create new secret key] 클릭
3. 이름 입력 → [Create] → 키 복사 (sk-...로 시작)
4. 💳 Billing에서 $5~10 충전 필요
5. 쇼츠 리믹서 [설정] → OpenAI API Key에 붙여넣기!
🚀 Groq · Llama 3.3 70B (무료, 매우 빠름)
1. console.groq.com/keys 접속 → 가입/로그인
2. [Create API Key] 클릭
3. 이름 입력 → 키 복사 (gsk_...로 시작)
4. 쇼츠 리믹서 [설정] → Groq API Key에 붙여넣기!
✅ 무료. 분당 호출 제한 있음

4. 💳 Billing에서 $5 충전 필요
